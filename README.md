## 三条线路到底有什么区别？不想踩坑必须看

说实话，很多人第一次看到 CN2 GIA、CMIN2、Tier 1 这几个词是懵的。把它翻译成人话：

**CN2 GIA（Pro 系列）**：中国电信最高端的国际专线，全程走 AS4809。晚高峰速度不掉、延迟不飙，这是它和普通线路最大的区别。代价是贵，而且如果你的用户以联通或移动为主，效果不如电信用户明显。

**CMIN2 / CMI（EB 系列）**：移动的国际优化线路，近年来口碑提升很快。洛杉矶 EB 系列三网回程走 CMIN2，联通用户和移动用户会觉得「跟 Pro 差不了多少」，但电信用户用 EB 系列体验会略逊于 Pro。价格和 Pro 相同，但流量配额更大。

**Tier 1（T1 系列）**：普通国际互联，不走电信联通的优化出口。国内高峰时段有可能感觉慢，但跑海外流量完全没问题。便宜，流量多，适合用途不需要国内优化的场景。
